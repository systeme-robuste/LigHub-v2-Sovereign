# LigHub v2.0 - NEXUS Sovereign Architecture

## Core Concept
Unified ecosystem for creators, voters, and researchers.

## Stack Technique (Target)
- **Frontend**: Next.js 14 (App Router), Tailwind CSS, Framer Motion.
- **Backend**: NestJS, PostgreSQL (Prisma), Redis.
- **AI Integration**: PulseGPT (OpenAI/Mistral/Local).
- **Security**: Double hashing system for votes, JWT + OAuth2.

## Project Structure
- `/backend`: NestJS API.
- `/frontend`: Next.js Web App.
- `/docs`: Architectural specs & API documentation.

## Current Phase: Core MVP (Phase 1)
- Secure Vote Engine.
- Markdown Content System.
- Basic Monetization (Stripe).
